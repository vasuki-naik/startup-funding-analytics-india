# Indian Startup Ecosystem Analysis & Funding Prediction

## Overview
This project analyzes startup funding trends in India using Python and Power BI.

## Tools
Python, Pandas, Matplotlib, Scikit-Learn, Power BI

## Key Insights
• Funding trends over years  
• Top funded industries  
• Most active investors  
• Investment type distribution  

## Files
startup_analysis.py → Data cleaning & analysis  
clean_startup_funding.csv → Processed dataset  
startup_dashboard.pbix → Power BI dashboard  
raw_startup_funding.xlsx → Original dataset
